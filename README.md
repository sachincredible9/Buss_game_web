# Bunny Learning | Official Hub

A modern, high-end landing page and centralized privacy hub for **Bunny Learning**. Designed with a premium "glassmorphism" aesthetic, this project is optimized for high-speed performance and long-term stability using simple HTML5 and Vanilla CSS.

## 🚀 Overview

This repository hosts the official single-page web application (SPWA) for Bunny Learning. It serves as the primary showcase for the app while providing a dedicated section for safety, transparency, and upcoming educational projects.

## 🔮 Key Features

- **Strict "Bunny Learning" Branding**: Consistent identity across all pages and documentation.
- **Premium Glassmorphism Aesthetic**: Minimalist design with vibrant child-friendly accents.
- **Privacy Hub**: Centralized section for COPPA/GDPR-K compliant privacy policies.
- **Parental Gate**: Built-in security for all external link redirects.
- **Performance Optimized**: Pure HTML/CSS for maximum speed and easy GitHub Pages hosting.

## 📂 Project Structure

- `index.html`: Main landing page for Bunny Learning.
- `css/main.css`: Core design system and glass effects.
- `js/main.js`: Interactive logic and safety gates.
- `privacy/`:
  - `index.html`: Master Privacy Hub index.
  - `bunny-learning.html`: Detailed policy for Bunny Learning.

## 🛠 Tech Stack

- **Structure**: HTML5
- **Styling**: Vanilla CSS
- **Logic**: Vanilla JavaScript
- **Hosting**: GitHub Pages

## ☁️ Deployment

To deploy to GitHub Pages:
1. Navigate to **Settings > Pages**.
2. Select the `main` branch.
3. Save and wait for the site to go live.
